Bibliotecas usadas

Flask: framework web (pip install Flask)

hashlib: biblioteca com implementação do sha256

Crypto: conforto na hora de exportar chaves RSA (pip install pycrypto)

criptography: RSA, hashes e para geração de documentos x509 
(pip install cryptography)

os, datetime, pathlib: para geração de diretórios dependentes do sistema 
operacional e datetime para especificação de data limite de validade
dos certificados



COMO RODAR O PROJETO:

usar o script do python no arquivo: python api.py
acessar o localhost no browser (http://127.0.0.1:5000/)
